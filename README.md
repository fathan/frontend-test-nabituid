## Getting Started

First, run the development server:

```bash
$ npm run dev
# or
$ yarn dev
# or
$ pnpm dev
```

For build to production run:
```bash
$ npm run build
```

For run Unit Testing:
```bash
$ npm run test
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
